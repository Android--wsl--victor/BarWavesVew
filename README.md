## 说明

1. 提供的 xml 属性

- barColor  横条颜色
- barHeight  横条高度
- waveRange  波浪条极差（最高与最低的差值）
- waveMinHeight  
- waveColor  波浪条最小高度
- waveWidth  波浪条颜色
- waveNumber  波浪条宽度
- waveInterval  波浪条数量
